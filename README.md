# 🛒 Market Lane

A modern React e-commerce storefront built with React, React Router, and Context API. Users can browse products, filter by category, search items, view product details, and manage a shopping cart with a complete bill summary.

---

## ✨ Features

- 🛍️ Browse all products
- 🔍 Search products
- 🏷️ Filter products by category
- 📄 Product details page
- 🛒 Shopping cart
- ➕ Add & remove products from cart
- ➖ Increase/Decrease item quantity
- 💰 Automatic bill summary
- 📱 Responsive UI
- ⚡ Fast routing using React Router
- 🌐 JSON-based product dataset

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/bhanurjb07/Tasks
```

### Navigate to the project

```bash
cd market-lane
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Open your browser at:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
